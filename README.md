\# AI Smart Assistant Pro



\## Description



AI Smart Assistant Pro is a Python-based conversational assistant developed during the Syntecxhub Artificial Intelligence Internship. The application can answer greetings, perform calculations, provide date and time information, remember user names, maintain chat history, tell jokes, display motivational quotes, and speak responses using text-to-speech technology.



\## Features



\* Greeting and Small Talk

\* Knowledge Base

\* Date and Time Commands

\* Calculator

\* User Memory

\* Chat History Logging

\* Jokes and Motivational Quotes

\* Colored Console Interface

\* Text-to-Speech Voice Output



\## Technologies Used



\* Python

\* Colorama

\* Pyttsx3

\* File Handling

\* Modular Programming
